# Exercise 6: JPA vs Hibernate vs Spring Data JPA

* **JPA (Java Persistence API):** It is just a specification (JSR 338) or a set of rules. It has NO implementation.
* **Hibernate:** It is the actual ORM framework and one of the most popular implementations of JPA. It does the heavy lifting of talking to the database.
* **Spring Data JPA:** It is an abstraction layer built on top of JPA/Hibernate. It drastically reduces boilerplate code by providing interfaces like JpaRepository.
